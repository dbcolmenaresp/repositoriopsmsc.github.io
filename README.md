# Directorio de Revistas y Sitios Académicos
Este directorio te permite explorar y filtrar una lista seleccionada de fuentes de información en ingeniería.

Navegación por Filtros: Utiliza los chips en la barra de filtros para ver solo Revistas, solo Sitios web, o para enfocarte en una disciplina específica como Ingeniería de Sistemas, Informática o Computación.

Búsqueda Rápida: El campo de búsqueda te permite filtrar las fuentes por nombre, editorial o palabras clave en la descripción. Presiona "Ir" o la tecla "Enter" para aplicar el filtro.

Información y Verificación: Cada tarjeta muestra el tipo de fuente, disciplina, año de fundación, una breve descripción y un enlace. Un indicador visual (verde "enlace válido" o rojo "no accesible") te informa sobre el estado de cada URL.
